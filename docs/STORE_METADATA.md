## 📖 전체 설명 (Play/App Store 공통, 4000자 제한)

```
🎨 Feeling Palette — 색으로 기록하는 AI 감정일기

오늘 당신의 기분은 어떤 색이었나요?
Feeling Palette는 당신의 하루 감정을 자유롭게 기록하고,
AI가 섬세하게 분석해주는 감정일기 앱입니다.


✨ 주요 기능

🖋️ 자유로운 기록
• 하루에 여러 번 기록 — 아침의 설렘, 저녁의 피로까지
• 간단한 텍스트 작성, 작성 시간 자동 표시
• 수정·삭제 언제든 가능

🤖 AI 감정 분석
• 기쁨·슬픔·분노·불안·평온·설렘 6가지 감정을 점수로 시각화
• 일기 내용에 맞춘 AI 코멘트 제공
• 일기당 최대 3회까지 재분석 가능

📅 감정 캘린더
• 한 달의 감정 흐름을 색으로 확인
• 여러 번 기록한 날은 평균 감정으로 대표 색 표시
• 날짜 선택 시 그날의 모든 기록과 감정 평균 그래프

📊 월간 통계
• 이번 달 가장 많이 느낀 감정 Top 3
• 감정 분포 도넛 차트
• 주간 감정 변화 선 그래프

📖 타임라인
• 작성한 모든 일기를 한눈에
• 무한 스크롤로 과거 기록도 쉽게 탐색

🔐 안전한 보관
• 4자리 PIN 잠금 + 지문·Face ID 인증
• 일기는 기기에만 저장됩니다
• Google Drive 개인 앱 폴더에 백업·복원 지원

💝 광고 없는 쾌적한 사용 (선택)
• 단 2,500원 일회성 결제로 배너·전면 광고 제거


💌 개인정보 보호를 최우선으로

Feeling Palette는 당신의 일기를 소중히 여깁니다.
• 일기 내용은 당신의 기기에만 저장됩니다
• AI 분석을 위해 일시적으로 서버 전송되지만 저장되지 않습니다
• 앱 잠금으로 타인의 접근을 차단합니다

자세한 내용: https://sedongcheon.github.io/feelingpalette-privacy/


📬 문의: sedong1000@gmail.com
```

글자 수: 약 600자 (한도 4000자 대비 충분한 여유).
