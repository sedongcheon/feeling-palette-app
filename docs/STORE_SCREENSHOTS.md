# 스토어 스크린샷 촬영 가이드

Google Play Store / Apple App Store에 올릴 스크린샷 준비. **실기기에서 30~40분** 정도 작업.

---

## 📐 해상도 요구사항

### Google Play Store
- 최소 **2장**, 권장 4~8장
- 크기: 16:9 또는 9:16, 최소 320px ~ 최대 3840px
- 형식: PNG 또는 JPEG
- 실기기 스크린샷 그대로 OK (보통 1080×2400 등)

### Apple App Store
- **iPhone 6.7"** (1290×2796) 최소 3장 **필수** — 최신 iPhone 기준
- iPhone 5.5" (1242×2208) 선택 — 구형 iPhone용
- iPad 있으면 별도 5장 더
- 형식: PNG (권장) 또는 JPEG

> 같은 스크린샷 세트로 Play + Apple 둘 다 커버 가능하지만, iOS 6.7"를 기준으로 찍으면 Play에도 그대로 쓸 수 있음 (더 큰 해상도 허용됨).

## 📋 촬영할 화면 5장 (추천 순서)

첫 번째 스크린샷이 가장 중요 — 검색 결과에 먼저 노출됨.

| # | 화면 | 보여줄 것 |
|---|------|----------|
| 1 | **홈 (작성 + 분석 결과)** | AI 감정 분석 결과 카드가 보이는 상태. "이게 뭘 하는 앱인지" 즉시 전달 |
| 2 | **감정 캘린더** | 여러 날이 감정 색으로 채워진 모습. 데이터 축적 가치 강조 |
| 3 | **월간 통계** | 도넛 차트 + Top3 + 주간 그래프. 인사이트 제공 |
| 4 | **홈 + 하루 여러 기록** | 같은 날 2~3개 기록 스택. 차별 기능 |
| 5 | **앱 잠금** (선택) | PIN 패드 or "보안" 강조. 개인정보 메시지 |

## 🎯 촬영 전 샘플 데이터 세팅

**중요**: 실제 개인 일기 내용이 스크린샷에 들어가지 않도록 **테스트 계정으로 새로 입력**하세요. 데이터 초기화 기능("비밀번호를 잊으셨나요?")으로 기존 데이터를 지운 뒤 진행하거나, 임시로 다른 디바이스/에뮬레이터 사용.

### 샘플 일기 텍스트 (복사해서 쓰세요)

**밝은 톤**
```
오랜만에 친구를 만나서 정말 즐거웠다. 카페에서 수다 떨면서 시간 가는 줄 몰랐어.
```

**중립/피곤**
```
오늘은 일이 많았다. 퇴근길 노을이 예뻐서 기분이 조금 나아졌지만 여전히 피곤하다.
```

**설렘**
```
다음 주 여행 계획을 세웠다. 벌써부터 설레고 기대된다.
```

**슬픔**
```
하루 종일 생각이 많았다. 아무것도 하기 싫은 날이었다.
```

**평온**
```
아침에 일찍 일어나 산책을 다녀왔다. 조용한 골목을 걸으며 마음이 편안해졌다.
```

### 세팅 순서
1. 앱 데이터 초기화 (또는 새 설치)
2. PIN 설정
3. **5~7일 전부터 오늘까지** 샘플 일기 각각 다른 감정으로 작성 → AI 분석 돌리기
4. 캘린더 탭에서 여러 색이 예쁘게 분포됐는지 확인
5. 오늘 날짜에 **여러 기록** 추가 (4번 화면용)
6. 통계 탭에서 Top3 잘 떴는지 확인

## ⚠️ 스크린샷에 광고가 보이면?

캘린더·통계·타임라인 탭 하단에 "Test Ad" 배너가 뜹니다. 두 가지 선택지:

### 옵션 A: 광고 가리기 (권장)
설정 → "광고 제거" 탭 → 테스트 환경에서는 결제 안 되지만, 개발자 모드에서 임시로 ad-free 설정 가능.

**또는 가장 간단**: `lib/services/ads_service.dart`에서 스크린샷 촬영 중만 `_adFree = true`로 임시 고정 (촬영 후 원복).

### 옵션 B: 배너 크롭
찍고 나서 배너 부분만 잘라내기. 앱 가로 폭은 유지한 채로 하단 50dp만 잘라냄. Preview.app / Keynote로 가능.

## 📸 실기기에서 스크린샷 찍기

### Android (Galaxy Z Fold5 등)
- 전원 + 볼륨 DOWN 동시에 짧게 누름
- 갤럭시는 엣지 스와이프 또는 손날 스크린샷도 가능
- 저장 위치: `DCIM/Screenshots/` → USB/Drive로 Mac에 전송

### iOS
- 전원 + 볼륨 UP 동시에 짧게 누름 (구형: 전원 + 홈)
- 좌하단 썸네일 탭 → 저장
- AirDrop으로 Mac에 바로 전송

### Flutter 개발 환경 (대안)
```bash
flutter screenshot --out=screen1.png
```
앱이 `flutter run` 중일 때 현재 화면을 PNG로 저장. 해상도는 디바이스 네이티브.
